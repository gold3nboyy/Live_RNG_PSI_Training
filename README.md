# Continuous Live Running Random Number Generator For PSI Training

It's a simple Random Number Generator that was nowhere to be found on the net, so I built one with Ai.
There are many generators that generate 1 number or a row of numbers etc. but nothing that goes on forever continuously.
This is where the real magic can begin.

Find your method of engagement and the right state of mind.
Apparently you can influence the output if done right.
Test your skills.

And even it doesn't work, this tool might help in other ways.
At least the internet has a free generator like this again (apparently there were some in the past, but not anymore).

I'm a complete and utter noob regarding all this, like hosting sites, programming stuff and the like.
So I'm not sure this site runs smoothly on your end like it does on mine.
I'm using a 1440p screen and hope the layout won't mess up on other resolutions.
I'm not sure it'll work on mobile phones, though. 

It took me days to get this running like it is now (v12.2), it's a good start.
But you're very welcome to to download and change whatever you like, host it, improve it and make it more usable or beautiful.

You'll find various functions and statistics. If you open up the hamburger menu you'll find detailed explanations to all of them. Once you start the generator 5 rows of 20 numbers appear below it: 1 solo number is always red; if you get 2 numbers of the same kind successively they'll turn orange; if you get 3 or more numbers of the same kind successively they'll turn green. That means if you manage to influence the generated output with a targeted number in mind you might be able to get a steak of green numbers! This could help visually as feedback.
